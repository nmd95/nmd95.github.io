# Nimrod's GitHub Pages starter

This folder is a minimal, static website styled like [tylerkastner.github.io](https://tylerkastner.github.io/) — plain HTML + CSS, no build steps.

## How to publish

1. Create a repository named `<your-username>.github.io`.
2. Upload the files in this folder to the repository root (including `.nojekyll`).
3. In the repo: **Settings → Pages → Source: Deploy from a branch** and choose `main` and `/ (root)`.
4. Visit `https://<your-username>.github.io`.

## Customize

- Replace `assets/portrait.jpg` with a real photo (same filename).
- Replace `/files/cv.pdf` with your CV (same filename). 
- Edit `index.html` text and links.
- Add more items to `publications/index.html`.

No frameworks used. If you want a theme (Jekyll), consider **Minimal Mistakes** or **al-folio**.
